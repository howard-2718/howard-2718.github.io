So then, let's get started on that proof! First, we'll consider the base case of \(n = 1\text{,}\) which is quite trivial. We see that

\begin{equation*} (a_{1} b_{1})^2 \leq a_{1}^2 b_{1}^2\text{,} \end{equation*}

which is obviously true. The crux of the proof lies in showing that

\begin{equation*} \left( \sum_{i = 1}^{k} a_{i} b_{i} \right)^2 \leq \left( \sum_{i = 1}^{k} a_{i}^2 \right) \left( \sum_{i = 1}^{k} b_{i}^2 \right) \implies \left( \sum_{i = 1}^{k+1} a_{i} b_{i} \right)^2 \leq \left( \sum_{i = 1}^{k+1} a_{i}^2 \right) \left( \sum_{i = 1}^{k+1} b_{i}^2 \right) \end{equation*}

holds for all \(k \in \mathbb{Z}^+\text{.}\)
